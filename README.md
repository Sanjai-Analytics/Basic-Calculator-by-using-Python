# 🧮 Python Calculator (Tkinter)

A simple calculator built using Python and Tkinter.  
It performs basic arithmetic operations with a user-friendly interface.

---

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear button
- Custom "Guvi" button
- PNG image as button

---

## 🛠️ Technologies Used
- Python
- Tkinter

---

## ▶️ How to Run

1. Install Python
2. Download or clone this repository
3. Open terminal / command prompt
4. Run:

python calculator.py

---

## 📁 Project Files

- calculator.py  → Main program
- logo.png       → Image button
